# Tawhidul Islam Ifty - Professional Portfolio Website

Welcome to my personal portfolio repository. This is a clean, modern, and fully responsive single-page portfolio application designed to showcase my skills, education, technical projects, achievements, and research work as a Computer Science student and Competitive Programmer.

---

## 🚀 Features

- **Interactive UI/UX:** Clean design with modern color schemes and a sticky dynamic navigation bar.
- **Smooth Navigation:** In-page smooth scrolling anchor links across 9 sections (Home, About, Education, Skills, Projects, Achievements, Publications, Hobbies, Contact).
- **Instant Resume Download:** Fully functional resume/CV download button directly linked to a structured professional local PDF resource.
- **Production-Ready Contact Form:** Fully configured server-side action handling powered by Formspree API integration.
- **Optimized Grid Layouts:** Clean and modular custom CSS grid system for displaying software projects seamlessly.

---

## 🛠️ Tech Stack Used

- **HTML5:** Semantic architecture layout structures.
- **CSS3:** Advanced flexbox architectures, multi-column grid containers, hover states, transitions, and unified styling schemas.
- **Git & GitHub:** For strict version control and workspace deployment workflows.
